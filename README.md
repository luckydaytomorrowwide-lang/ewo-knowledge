# test

このディレクトリは  
EWO Factory 作業で使用する **ファイル受け渡し場所**です。

ユーザーが AI に提供するファイル、  
および EWO生成・実行・解析に必要な資材を保存します。

試験資材だけでなく、  
EWO生成に関係するあらゆるファイルを保存することがあります。

---

# 目的

EWO Factory 作業に必要なファイルを  
ユーザーと AI の間で共有すること。

主な用途

- EWO生成に必要な入力ファイル
- workflow実行資材
- テストデータ
- 実行ログ
- 解析対象データ

---

# 使用工程

このディレクトリは **PLAN工程から使用します。**

PLAN  
EWO生成のための入力ファイル取得

DO  
workflow実行資材

CHECK  
ログ解析資材

---

# ファイル提供

ユーザーは  
EWO生成または解析に必要なファイルを  
このディレクトリに配置します。

AIはこのディレクトリのファイルを使用して  
EWO生成・実行・解析を行います。

---

# ファイル名

ファイル名は **任意です。**

例

- EWO.json
- input.json
- table.json
- layout.json
- data.json
- log.json

これらは一例であり、  
必ずしもこの名前である必要はありません。

---

# 保存されるファイル

このREADMEに記載されていないファイルが  
保存される場合があります。

例

- CSV
- XLSX
- JSON
- ログ
- 中間データ

---

# 不明なファイル

AIがファイルの用途を判断できない場合は  
推測して使用してはいけません。

ユーザーに質問し、用途を確認します。

---

# 注意

このディレクトリのファイルは  
EWO生成・実行・解析の再現性を保つために保存されます。

AIはファイルの内容を確認し、  
必要な資材を特定して作業を進めます。
