# EWO Factory Assistant - Entry Point

チャット開始時は **必ずこのページを最初に参照すること。**

このリポジトリは **EWO Factory の知識ベース（SSOT）**である。

EWO生成・実行・解析・改善は GitHub上の情報を根拠として行う。推測は禁止。

------------------------------------------------------------------------

# 1. 依頼内容を理解する

ユーザーの依頼が何を求めているかを理解する。

例 - workflowの説明 - EWO作成 - workflow実行 - ログ解析 - 修正

------------------------------------------------------------------------

# 2. 質問タイプを判定する

  質問タイプ   内容
  ------------ ---------------------
  構造理解     workflowやEWOの説明
  生成         新しいEWO作成
  実行         workflow実行
  解析         結果やログ解析
  改善         修正や改善

------------------------------------------------------------------------

# 3. 工程を判定する

  質問タイプ   工程
  ------------ --------
  構造理解     PLAN
  生成         PLAN
  実行         DO
  解析         CHECK
  改善         ACTION

------------------------------------------------------------------------

# 4. 工程ガイドを参照する

  工程     参照ファイル
  -------- --------------
  PLAN     plan.md
  DO       do.md
  CHECK    check.md
  ACTION   action.md

------------------------------------------------------------------------

# 5. GitHub参照

spec → decisions → examples → templates → checklists

spec が最優先。
